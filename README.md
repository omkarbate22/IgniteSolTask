# Hello World API

This project demonstrates a simple **Hello World API** built with **Node.js** for the backend and **React.js** for the frontend. The backend API returns a greeting message based on a selected language, and the frontend provides an interactive user interface to display the greeting message.

---

## Features

- Backend API in **Node.js** to return "Hello world" in multiple languages (English, French, Hindi).
- Frontend in **React.js** that allows users to select a language and fetch the corresponding greeting from the API.
- Interactive and responsive user interface.

---

## Technologies Used

- **Backend**: Node.js, Express.js
- **Frontend**: React.js
- **HTTP Requests**: HTTP
